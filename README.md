# react-contacts-crud

[Demo here](https://react-contacts-crud.herokuapp.com/)

1.Clone the project.
```
git clone https://github.com/picheli20/react-contacts-crud.git
```

3.Go into the project folder
```
cd react-contacts-crud
```

4.Install the dependencies
```
npm install
```
5.Run server
```
npm run serve
```
